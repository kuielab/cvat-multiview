# CVAT 단축키 가이드

## Track 라벨링

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `O` | Outside 전환 | 객체가 화면에서 사라짐/나타남 표시 |
| `K` | 키프레임 설정/해제 | 현재 프레임을 키프레임으로 지정 |
| `Delete` | 객체 삭제 | 선택한 객체 삭제 |
| `Shift+Delete` | 잠긴 객체 강제 삭제 | Lock된 객체도 삭제 |

---

## 네비게이션

| 단축키 | 기능 |
|--------|------|
| `D` | 다음 프레임 |
| `A` | 이전 프레임 |
| `V` | 앞으로 10프레임 |
| `C` | 뒤로 10프레임 |
| `Space` | 재생/일시정지 |

---

## 객체 조작

| 단축키 | 기능 |
|--------|------|
| `Ctrl+C` | 객체 복사 |
| `Ctrl+V` | 객체 붙여넣기 |
| `Ctrl+Z` | 실행 취소 |
| `Ctrl+Shift+Z` | 다시 실행 |
| `Delete` | 선택한 객체 삭제 |
| `F` | 객체를 앞으로 (z-order) |
| `B` | 객체를 뒤로 (z-order) |

---

## Draw 모드

| 단축키 | 기능 |
|--------|------|
| `N` | 새 객체 그리기 시작 |
| `Esc` | 그리기 취소 |
| `Enter` | 폴리곤/폴리라인 완료 |

---

## 뷰 조작

| 단축키 | 기능 |
|--------|------|
| `Ctrl+마우스휠` | 줌 인/아웃 |
| `Alt+드래그` | 캔버스 이동 (Pan) |
| `Ctrl+0` | 줌 리셋 (Fit to window) |

---

## 라벨 선택

| 단축키 | 기능 |
|--------|------|
| `1-9` | 라벨 1-9 선택 |
| `0` | 라벨 10 선택 |

---

## 기타

| 단축키 | 기능 |
|--------|------|
| `Ctrl+S` | 저장 |
| `Ctrl+A` | 모든 객체 선택 |
| `Tab` | 다음 객체 선택 |
| `Shift+Tab` | 이전 객체 선택 |
| `H` | 선택한 객체 숨기기/보이기 |
| `Ctrl+H` | 모든 객체 숨기기/보이기 |

---

## Track Outside 사용 예시

1. 객체가 화면에서 **사라지는 마지막 프레임**으로 이동
2. Track 선택
3. `O` 키 → Outside 활성화 (점선으로 표시됨)
4. 객체가 **다시 나타나는 프레임**으로 이동
5. `O` 키 → Outside 해제
6. 위치 조정
